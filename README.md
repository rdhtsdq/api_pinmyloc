# Api Pinmyloc

backend for pinmyloc

## Installation

```bash
git clone https://github.com/rdhtsdq/api_pinmyloc.git
npm install
```

## Usage

```bash
#for development
npm run dev

#for production
npm start
```
for production make sure to install [pm2](https://www.npmjs.com/package/pm2) package
